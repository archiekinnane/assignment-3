## Assignment 3
This repository contains Python scripts for database operations and data querying.

### Files
create_db.py: Script to create the database and its tables.
insert_data.py: Script to insert sample data into the database.
query.py: Script to run queries against the database and display results.
output.txt: The desired output of query.py.

### Usage
Run create_db.py to set up the database.
Run insert_data.py to populate the database with data.
Use query.py to execute queries and view results.

### Requirements
sqlite3
